linux最最基本指令

linux最最基本指令

2025-02-02
阅读时长: 2 分钟

linux最最基本指令及英文全称(由来)

遨游于文件系统

pwd (print working directory)

  • 语法 pwd

  • 作用 输出当前工作目录

cd (change directory)

  • 语法 cd [target path]
    例如
1

cd ./home/linlu2cath/Documents
  • 作用 切换目录至 [ target path ]

ls (list)

  • 语法 ls [directory] or ls (对象为当前目录)
    例如
1

ls ./home/linlu2cath/Documents
  • 作用 列出 [ directory ] 下所有文件
  • 常用选项 ls -l( long )
    列出 [ directory ] 下所有文件及文件属性
    ls -a( all )
    列出 [ directory ] 下所有文件及隐藏文件

clear (clear)

  • 语法 clear
  • 作用 清除屏幕
  • 快捷方式 ctrl+l

xdg-open (x-desktop-group open)

  • 语法 xdg-open [file]
  • 作用 打开 [ file ] ( 以适当的软件打开 )

将文件玩弄于股掌之中

cp (copy)

  • 语法 cp [source] [destination]
  • 作用 将 [ source ] 复制至 [ destination ]

mv (move)

  • 语法 mv [source] [destination]
  • 作用 将 [ source ] 移动至 [ destination ]

rm (remove)

  • 语法 rm [file]
  • 作用 删除 [ file ]

touch (touch)

  • 语法 touch [file]
  • 作用 创建 [ file ]或更新 [ file ]的修改时间( 3touch这个文件,如果没有这个文件,则创建这个文件)

cat (concatenate)

  • 语法 cat [file]
  • 作用 输出 [ file ]

mkdir (make directory)

  • 语法 mkdir [directory]
  • 作用 创建 [ directory ]

rmdir (remove directory)

  • 语法 rmdir [directory]
  • 作用 删除 [ directory ] ( 如果目录不为空,则删除失败 )
  • 常用选项 rmdir [directory] -r( recursive )
    递归删除 [ directory ] 及其子目录及文件

tail (tail)

  • 语法 tail [file]
  • 作用 输出 [ file ] 的最后10行
  • 常用选项 tail -n [number] [file] 输出 [ file ] 的最后 [ number ] 行

流

<

  • 语法 [command] < [file]
  • 作用 将 [ file ] 的内容作为 [ command ] 的输入流

>

  • 语法 [command] > [file]
  • 作用 将 [ command ] 的输出流作为 [ file ] 的内容

»

  • 语法 [command] >> [file]

  • 作用 将 [ command ] 的输出流作为 [ file ] 的内容,并追加内容

管道( 将左边的内容流向右边 )

  • 语法 [source] | [destination]
  • 作用 将 [ source ] 的输出流流向 [ destination ] 的输入流
  • 实例 ll | tail -n1 > ls.txt

tee (tee)

  • 语法 [command] | tee [file]
  • 作用 将 [ command ] 的输出流流向 [ file ] 的输入流,并输出到屏幕
  • 实例 ll | tee ls.txt

用户与权限

sudo (super user)

  • 语法 sudo [command]
  • 作用 以管理员身份执行 [ command ]
  • 常用选项 sudo -i( interactive )
    以管理员身份进入当前用户

chmod (change mode)

  • 语法 chmod [permission] [file]

  • 作用 修改 [ file ] 的权限

  • 实例

chmod 751 [file] ( ( change every single decNum to biNum ) 修改权限为 rw-r-x–x )
chmod u+x [file] ( 为 user 添加执行权限 )
chmod o-w [file] ( 为 other 移除写入权限 )

chown (change owner)

  • 语法

chown [owner]:[group] [file]

  • 作用 修改 [ file ] 的拥有者为 [ owner ]
  • 实例 chown linlu2cath:exampleGroup exampleFile.txt

exit (exit)

  • 语法 exit
  • 作用 退出当前用户
